FashionX merupakan sebuah website e-commerce fashion yang dikembangkan sebagai project pembelajaran pada mata kuliah Matematika Diskrit. Website ini tidak hanya menampilkan fitur transaksi jual beli produk fashion, tetapi juga mengimplementasikan konsep-konsep Matematika Diskrit dalam sistemnya, khususnya pada aspek keamanan dan pengolahan data.

## Tujuan Pengembangan
Tujuan dari pengembangan website FashionX adalah:
1. Menerapkan konsep Matematika Diskrit ke dalam sistem e-commerce berbasis web.
2. Mengimplementasikan kriptografi sederhana sebagai simulasi keamanan data.
3. Membangun alur transaksi digital yang terstruktur dan logis.
4. Menggabungkan aspek fungsionalitas sistem dengan tampilan antarmuka yang ramah pengguna.

## Konsep Matematika Diskrit yang Digunakan
Website ini mengimplementasikan beberapa konsep Matematika Diskrit, antara lain:
- Himpunan, untuk pengelompokan pengguna, produk, dan kategori.
- Fungsi hash, untuk simulasi pengamanan data pengguna.
- Hill Cipher, untuk proses enkripsi data transaksi dan invoice.
- Logika matematika, untuk validasi proses checkout dan verifikasi OTP.
- Graf, untuk pemodelan alur navigasi dan hubungan antar halaman.

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- React
- Tailwind CSS
- LocalStorage sebagai simulasi basis data

## Fitur Utama Sistem
- Homepage dan katalog produk fashion
- Keranjang belanja
- Proses checkout dan simulasi pembayaran
- Sistem login dan verifikasi OTP
- Pembuatan invoice transaksi
- Halaman riwayat pesanan pengguna

## Catatan Pengembangan
Website ini dikembangkan sebagai media pembelajaran. Seluruh sistem pembayaran dan keamanan bersifat simulasi dengan tujuan memahami konsep logika, struktur data, dan Matematika Diskrit dalam pengembangan perangkat lunak.
